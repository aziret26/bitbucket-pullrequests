Current project has been developed according to Zensoft Test Assignment.
Purpose of the project to parse json data retrieved from bitbucket.org repository,
and print list of pull requests.

----------

How to use.

Main class to run: PullRequestRetriever.
Main class class takes single argument: bitbucket "repository name"
example for bitbucket "repository name" : "aziret26/first-repo".

If no repository is provided program asks for one.

---------

Example:
java PullRequestRetriever aziret26/first-repo
